# Zhejie Jiang | Personal Portfolio
Welcome to my personal portfolio website!

## Features

- **Starry background**: Falling star animation on the homepage.
- **Bubble effect**: Floating bubbles on the homepage (TypeScript + CSS).
- **Nav active state**: Current page is highlighted in the navbar (set via TypeScript).
- **Button and link polish**: Hover scale/glow, focus-visible outlines, active press feedback.
- **About Me**: GitHub contributions chart and bio.
- **Projects**: Project cards with hover lift and glow.
- **Contact**: Form and social links.
- **Social links**: LinkedIn, Instagram, GitHub.
- **Dark mode**: Toggle in navbar; preference saved in localStorage and respects system preference on first visit.
